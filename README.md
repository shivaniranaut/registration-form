# ✨ Aesthetic Registration Form

A beautifully designed registration form built with pure **HTML & CSS** — featuring glassmorphism, smooth animations, and an orange glow focus effect.

![Registration Form Preview](preview.png)

---

## 🚀 Live Demo

Open `index.html` directly in your browser — no setup needed!

---

## ✨ Features

- 🌫️ **Glassmorphism card** — blurred dark transparent background
- 🔶 **Orange glow focus effect** — inputs glow on click
- 🎞️ **Slide-up animation** on page load
- 🖱️ **Hover effect** on submit button with lift + glow
- 🌄 **Scenic background image** — road trip desert aesthetic
- 📱 **Fully responsive** — works on mobile and desktop
- 💯 **Pure HTML & CSS** — zero JavaScript, zero libraries

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, Grid, Backdrop Filter, Animations)
- Google Fonts (Playfair Display + DM Sans)
